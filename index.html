Making an Operating system.

Here in this tutorial/guide, we are going to create a simple boot program (O.S) which displays a "hello world" message then hangs.

Source Code

; A simple boot program that prints a message to the screen.
mov ah , 0 x0e ; int 10/ ah = 0eh -> scrolling teletype BIOS routine
mov al , 'H'
int 0x10
mov al , 'e'
int 0x10
mov al , 'l'
int 0x10
mov al , 'l'
int 0x10
mov al , 'o'
int 0x10
jmp $ ; Jump to the current address forever.
;
times 510 -($-$$) db 0 ; Pad the boot sector out with zeros
dw 0 xaa55 ;

compiling

Open or launch command prompt. Copy and paste the following code in it. nasm boot.asm -o boot.bin
